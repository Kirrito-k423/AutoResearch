# Phase 08 Plan 01: minimal-runner 抽象 + verl/veomni 实例 (D-44)

D-44 锁: 最小实验复用 Phase 7 1-step 脚本 (D-41 torch_npu), 跨 A2-AK-225 已知 OK. 本 plan 把 runner 抽到
`verl-workspace-adapter/`, 然后 `autoresearch/collect/minimal.py` 编排跑一次.

## Tasks

1. **`verl-workspace-adapter/common/conda_utils.py`**
   - `run_in_env(spec: ServerSpec, workdir: str, command: str, timeout: float) -> tuple[int, str, str]`
   - 走 `cd <workdir> && (conda run -n <env> <command> | python -c)`, 单次干净
   - workdir 走 cd, 不动 ssh 客户端状态
2. **`verl-workspace-adapter/verl/minimal_runner.py`**
   - `ONE_STEP_SCRIPT` 与 Phase 7 `ONE_STEP_SCRIPT_TMPL` 同步 (D-41 torch_npu, lib=verl)
   - `run_minimal(spec: ServerSpec, workdir: str, lib: str = "verl", conda_env: str = "", timeout: float = 30.0) -> MinimalResult`
   - `MinimalResult` TypedDict: `{sum_value, npu_count, elapsed_ms, exit_code, stdout, stderr, error}`
   - 解析 SUM= / NPU_COUNT= 同 Phase 7 parser
3. **`verl-workspace-adapter/veomni/minimal_runner.py`**
   - 同结构, lib="veomni", 复用 `ONE_STEP_SCRIPT` 模板 (lib 占位)
4. **`autoresearch/collect/minimal.py`**
   - `collect_minimal(server: str, lib: str, config_path=None, workdir_override=None) -> MinimalResult`
   - 读 config → 找 spec → 调对应 runner (`verl` → `verl-workspace-adapter.verl.minimal_runner`, `veomni` → `veomni.minimal_runner`)
   - 返回 MinimalResult 给下游 (08-02 wandb sync 读 wandb_run_id 还没出来, 留 08-04 manifest)
5. **测试**:
   - `tests/test_minimal_runner.py`: mock SSHClient 测 4 路径 (PASS / timeout / exit!=0 / no-SUM)
   - `tests/test_collect_minimal.py`: mock runner 测 collect_minimal 派发 (verl / veomni)
6. **UAT** (`08-01-SUMMARY.md`):
   - 1 台真机 (A2-AK-225) 跑 `autoresearch collect minimal --server A2-AK-225 --lib verl`, 验 MinimalResult 含 SUM + NPU_COUNT
   - 不接 wandb sync / log / prom (08-02..08-04), 只验 runner 本身

- **R1 conda_env 缺省**: D-46 workdir 字段 08-01 还没加 schema, 08-01 临时读 config 然后走 spec.host + spec.user 直连, workdir 从 spec.workdir 读 (新字段). 04 写 schema 时一并加.
- **R2 verl/veomni 框架包 11s 冷启**: 1-step 30s timeout 同 Phase 7 (D-41)
- **R3 lib 字符串硬编码**: verl / veomni 之外 (D-43 决策) 暂不支持, v1.1 扩
