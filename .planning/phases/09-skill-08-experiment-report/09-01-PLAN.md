# Phase 09 Plan 01: 读 manifest + 收集三路数据

先把 `run_id -> ReportBundle` 这条链打通：manifest、log、wandb、prom 都整理成统一数据结构，后面的 HTML 渲染只吃 bundle，不再散落地读文件和拼链接。

## Tasks

1. **`autoresearch/report/models.py`**
   - 定义 `ReportBundle` / `LogView` / `WandbView` / `PrometheusView` / `ReportWarning` 等轻量模型
   - 约定 partial 场景：每一路都能独立标 `available / warning / error / source_path / source_url`
   - `ReportBundle` 至少含：
     - run 元信息：`run_id, server, lib, conda_env, started_at, finished_at, exit_code, error`
     - log 摘要
     - wandb 摘要
     - Prometheus 摘要
     - raw artifact links

2. **`autoresearch/report/logs.py`**
   - 从 `manifest.log_files[0]` 读取本地 `log.txt`
   - 提取关键行（优先 `SUM=`, `NPU_COUNT=`, `WANDB_RUN_ID=`, `wandb:` summary）
   - 同时保留末尾片段（例如最后 20-40 行）供 HTML 渲染
   - 文件缺失时不抛硬错误，转成 warning

3. **`autoresearch/report/wandb.py`**
   - 从 `manifest.wandb_path/files/wandb-summary.json` 读取最小指标
   - 组装本地 wandb 链接信息：
     - deep-link 若能稳定构造
     - 保底 root URL `http://localhost:8080/` + `wandb_run_id`
   - 当前 M1 只有单点数据，因此输出 snapshot metrics + 单点图所需数值，不追求历史曲线解析

4. **`autoresearch/report/prometheus.py`**
   - 基于 `run_id` 构造固定查询：
     - `autoresearch_npu_count{run_id="<run_id>"}`
   - 优先尝试本地 `http://localhost:9090/api/v1/query` 或 `query_range`
   - 无法连接或无数据时，保留 query URL 和 `prom_pushed` 状态，不阻塞 report

5. **`autoresearch/report/loader.py`**
   - `load_report_bundle(run_id: str, root: Path | None = None) -> ReportBundle`
   - 统一流程：读 manifest -> 读 log -> 读 wandb -> 读 prom -> 合并 warnings / links
   - 复用 `workspace-core/layout/paths.py` 找 run 根目录，避免手写路径拼接

6. **测试**
   - `tests/test_report_loader.py`
     - happy path: manifest + 三路数据齐，输出完整 bundle
     - partial path: 缺 log / 缺 wandb / prom 不可达时仍返回 bundle
   - `tests/test_report_wandb.py`
     - `wandb-summary.json` 解析
     - root link / deep-link 回退
   - `tests/test_report_prometheus.py`
     - query URL 构造
     - 本地 Prometheus 不可达时 graceful fallback

## 风险

- **R1 wandb deep-link 路由不稳定**：Plan 01 先把 root link + `wandb_run_id` 保底打通；deep-link 作为 best-effort，不把整个 phase 卡死。
- **R2 Prometheus 当前只有单点值**：这不是 bug，是 D-53 锁定的数据粒度；实现必须用 snapshot 语义表达。
- **R3 外部服务瞬时不可达**：bundle 层要把失败吸收成 warning，不能让 HTML render 没输入。
