# PP

- **参数名**：`PP`
- **分类**：效率
- **中文解释**：文档说明：examples 中用于设置 Megatron `pipeline_model_parallel_size` 的环境变量，即 actor/ref 训练侧流水线并行度。
- **常见值**：1、2、8
- **来源环境变量**：PP
- **性能影响**：文档说明：PP 可降低单卡承载的层数和显存压力；Verl best practices 建议在必要时增加 PP，但要平衡流水线气泡、通信和调度开销。
- **精度影响**：机制推断：流水线并行不改变训练目标；但微批划分、通信顺序和混合精度归约可能造成轻微可复现性差异。
- **NPU/Ascend 证据**：未知
- **CI 看护**：部分
- **示例数**：6
- **需要子代理补证**：否
