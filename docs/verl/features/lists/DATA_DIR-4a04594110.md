# DATA_DIR

- **参数名**：`DATA_DIR`
- **分类**：配置
- **中文解释**：机制推断：示例脚本中的数据根目录，GRPO/GDPO 等脚本用它派生 `data.train_files`、`data.val_files` 或本地数据集路径。
- **常见值**："$PWD"、$HOME/data/rlla_4k
- **来源环境变量**：DATA_DIR
- **性能影响**：机制推断：只改变数据文件定位，通常不影响单步吞吐；慢盘、远程挂载或路径错误会影响加载耗时和作业可靠性。
- **精度影响**：机制推断：目录本身不改训练目标；若目录下的数据集、切分或清洗版本不同，会直接改变训练样本分布和验证结果。
- **NPU/Ascend 证据**：未知
- **CI 看护**：部分
- **示例数**：2
- **需要子代理补证**：否

## 示例脚本

- `examples/gdpo_trainer/run_qwen3_8b_fsdp.sh`
- `examples/grpo_trainer/run_qwen3_30b_a3b_megatron.sh`

## 证据片段

- `examples/grpo_trainer/run_qwen3_30b_a3b_megatron.sh:18` DATA_DIR=${DATA_DIR:-"$PWD"}
- `examples/gdpo_trainer/run_qwen3_8b_fsdp.sh:12` DATA_DIR=${DATA_DIR:-$HOME/data/rlla_4k}
