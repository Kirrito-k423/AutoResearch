# CP

- **参数名**：`CP`
- **分类**：效率
- **中文解释**：文档说明：examples 中的 `CP` 环境变量通常映射到 Megatron `context_parallel_size`，即上下文并行度；用于在长序列训练中沿序列/上下文维度切分计算。
- **常见值**：1、2
- **来源环境变量**：CP
- **性能影响**：文档说明：CP 可扩展长序列容量并降低单卡激活显存，但会增加跨设备通信；需要与 TP/PP/EP、micro batch 和硬件拓扑一起平衡。
- **精度影响**：机制推断：并行切分正确时应保持数学等价；不同通信/归约顺序可能造成微小非确定性，配置不当会 OOM 或通信失败而影响训练可用性。
- **NPU/Ascend 证据**：未知
- **CI 看护**：部分
- **示例数**：4
- **需要子代理补证**：否
