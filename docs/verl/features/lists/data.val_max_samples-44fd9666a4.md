# data.val_max_samples

- **参数名**：`data.val_max_samples`
- **分类**：效率
- **中文解释**：文档说明：验证集最多使用的样本数；官方 config 文档说明设为 `-1` 表示使用完整验证集，示例用 1024 控制评测规模。
- **常见值**：1024
- **来源环境变量**：无
- **性能影响**：文档说明：限制验证样本可显著降低 eval 时间和显存/数据加载成本，尤其在频繁 `test_freq` 下；全量验证更慢但统计更充分。
- **精度影响**：机制推断：不直接影响训练更新，但会影响验证指标的方差和代表性；样本过少可能让模型选择、早停或回归判断偏噪。
- **NPU/Ascend 证据**：未知
- **CI 看护**：未知
- **示例数**：1
- **需要子代理补证**：否
