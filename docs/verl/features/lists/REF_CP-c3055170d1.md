# REF_CP

- **参数名**：`REF_CP`
- **分类**：效率
- **中文解释**：文档说明：reference policy 的 Megatron context parallel size，示例写入 `actor_rollout_ref.ref.megatron.context_parallel_size`，用于参考模型 logprob 计算侧的上下文并行切分。
- **常见值**：1
- **来源环境变量**：REF_CP
- **性能影响**：机制推断：增大 CP 可分摊长序列激活/注意力相关显存，但会带来额外通信和并行调度开销。
- **精度影响**：机制推断：正确并行切分应保持 reference logprob 等价；不兼容或切分错误会导致数值差异、同步错误或运行失败。
- **NPU/Ascend 证据**：未知
- **CI 看护**：部分
- **示例数**：1
- **需要子代理补证**：否

## 示例脚本

- `examples/grpo_trainer/run_qwen3_30b_a3b_megatron.sh`

## 证据片段
