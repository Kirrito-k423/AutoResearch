# TAU_POS

- **参数名**：`TAU_POS`
- **分类**：效率
- **中文解释**：源码说明：SAPO 策略损失中正优势 token 使用的 tau 参数；当 advantage 大于 0 时，源码用 `tau_pos` 进入门控函数来调节 importance ratio 的更新权重。
- **常见值**：1.0
- **来源环境变量**：TAU_POS
- **性能影响**：机制推断：只是损失函数中的标量超参，额外计算成本可忽略；不直接影响显存或分布式通信。
- **精度影响**：源码说明：直接改变正优势样本的策略梯度门控曲线；影响好动作概率提升的幅度与稳定性，是 SAPO 训练动态参数。
- **NPU/Ascend 证据**：未知
- **CI 看护**：部分
- **示例数**：2
- **需要子代理补证**：否
