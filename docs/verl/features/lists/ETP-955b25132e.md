# ETP

- **参数名**：`ETP`
- **分类**：效率
- **中文解释**：文档说明：`ETP` 是 examples 中用于填充 Megatron `expert_tensor_parallel_size` 的环境变量，控制 MoE 专家内部的张量并行度；Verl 参数表把它解释为“专家 TP 并行大小”。
- **常见值**：1
- **来源环境变量**：ETP
- **性能影响**：文档说明：Verl 性能建议要求 PP/TP/EP/ETP/CP 按显存和网络约束平衡；增大 ETP 可降低单卡专家权重/激活压力，但会增加专家内部通信和拓扑调优成本。
- **精度影响**：机制推断：并行切分本身通常不改变训练目标；仅可能因不同通信归约顺序或 kernel 后端带来极小数值差异，主要风险仍是配置不匹配导致训练失败。
- **NPU/Ascend 证据**：未知
- **CI 看护**：部分
- **示例数**：4
- **需要子代理补证**：否
