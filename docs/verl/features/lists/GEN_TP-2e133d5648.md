# GEN_TP

- **参数名**：`GEN_TP`
- **分类**：效率
- **中文解释**：文档说明：examples 中用于给生成/rollout 后端设置 `actor_rollout_ref.rollout.tensor_model_parallel_size` 的环境变量，即推理侧 tensor parallel 度。
- **常见值**：16、4、8
- **来源环境变量**：GEN_TP
- **性能影响**：文档说明：提高 rollout TP 可扩大单次推理可用显存和 KV cache 容量，但会增加通信成本；Verl best practices 提醒 TP 较大，尤其大于 8 时要关注通信开销。
- **精度影响**：机制推断：并行度本身不改变采样目标；但如果 TP 太小导致 OOM、长度下调或 backend 切换，会间接改变训练/评测口径。
- **NPU/Ascend 证据**：未知
- **CI 看护**：部分
- **示例数**：7
- **需要子代理补证**：否
