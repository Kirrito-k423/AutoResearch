# RANK

- **参数名**：`RANK`
- **分类**：效率
- **中文解释**：机制推断：多节点 `torchrun` 的节点 rank，示例作为 `--node_rank` 使用，用来标识当前节点在分布式作业中的序号。
- **常见值**：0
- **来源环境变量**：RANK
- **性能影响**：机制推断：正确配置后不直接影响吞吐；rank 重复或错配会导致分布式通信初始化失败、挂起或节点角色混乱。
- **精度影响**：机制推断：不属于精度调节参数；错误配置通常导致作业无法正常运行，而不是稳定改变模型质量。
- **NPU/Ascend 证据**：未知
- **CI 看护**：部分
- **示例数**：1
- **需要子代理补证**：否

## 示例脚本

- `examples/sft/gsm8k/run_mimo_7b_mtp_megatron.sh`

## 证据片段
