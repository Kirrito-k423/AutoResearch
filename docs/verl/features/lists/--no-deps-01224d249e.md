# --no-deps

- **参数名**：`--no-deps`
- **分类**：效率
- **中文解释**：文档说明：pip 安装参数，跳过依赖安装；Verl 官方安装说明在已有基础镜像或预装依赖时用它安装 Verl/Megatron 相关包，避免覆盖环境中的固定依赖版本。
- **常见值**：git+https://github.com/NVIDIA-NeMo/Megatron-Bridge、git+https://github.com/NVIDIA/Megatron-LM.git@core_dev_r0.16.0
- **来源环境变量**：无
- **性能影响**：机制推断：可缩短安装时间并减少依赖解析/下载，但要求环境已满足依赖；缺失或版本不兼容会导致启动失败或算子退化。
- **精度影响**：机制推断：安装参数不直接影响训练精度；若跳过依赖导致实际运行库版本与预期不同，可能间接影响数值一致性或稳定性。
- **NPU/Ascend 证据**：未知
- **CI 看护**：部分
- **示例数**：2
- **需要子代理补证**：否
