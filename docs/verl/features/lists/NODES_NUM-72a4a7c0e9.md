# NODES_NUM

- **参数名**：`NODES_NUM`
- **分类**：效率
- **中文解释**：机制推断：示例中的多节点规模参数，写入 `trainer.nnodes`，表示训练任务使用的节点总数。
- **常见值**：4
- **来源环境变量**：NODES_NUM
- **性能影响**：机制推断：增大节点数可提供更多设备和更大模型/批量承载能力，但会增加跨节点通信、同步和调度开销；设置错误会导致资源不足或分布式启动失败。
- **精度影响**：机制推断：节点数本身不改变算法目标；若同时改变全局 batch、随机种子或样本并行顺序，可能间接影响训练可复现性和收敛轨迹。
- **NPU/Ascend 证据**：未知
- **CI 看护**：未知
- **示例数**：1
- **需要子代理补证**：否
