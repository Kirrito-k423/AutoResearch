# NNODES

- **参数名**：`NNODES`
- **分类**：效率
- **中文解释**：示例脚本中的节点数环境变量，通常传入 `trainer.nnodes` 或启动脚本，用于声明多节点训练规模。
- **常见值**：1、12、16、2、3、4、6、8
- **来源环境变量**：NNODES
- **性能影响**：文档说明：官方多节点文档用节点数启动 Ray head/worker 并设置 `trainer.nnodes`；增加节点可扩大总设备数，但跨节点通信、Ray 调度和网络带宽会成为性能变量。
- **精度影响**：机制推断：作为资源变量不直接改变模型目标；只有间接改变全局 batch、并行划分或重试行为时才可能影响结果。
- **NPU/Ascend 证据**：部分
- **CI 看护**：部分
- **示例数**：67
- **需要子代理补证**：否
