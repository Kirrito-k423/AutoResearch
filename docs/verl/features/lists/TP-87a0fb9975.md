# TP

- **参数名**：`TP`
- **分类**：效率
- **中文解释**：文档说明：examples 中用于设置 Megatron `tensor_model_parallel_size` 的环境变量，即 actor/ref 训练侧张量并行度。
- **常见值**：2、4
- **来源环境变量**：TP
- **性能影响**：文档说明：增大 TP 可降低每卡参数/激活压力；Verl best practices 指出 bf16/fp16 下每个参数约按 `2 / TP` 字节分摊，但 TP 增大会带来通信开销。
- **精度影响**：机制推断：张量并行不改变目标函数；不同切分和归约顺序可能带来轻微数值差异，主要风险来自资源不足时被迫调小 batch/长度。
- **NPU/Ascend 证据**：未知
- **CI 看护**：部分
- **示例数**：6
- **需要子代理补证**：否
