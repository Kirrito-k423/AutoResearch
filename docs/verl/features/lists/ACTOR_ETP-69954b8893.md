# ACTOR_ETP

- **参数名**：`ACTOR_ETP`
- **分类**：效率
- **中文解释**：文档说明：examples 中用于设置 actor Megatron `expert_tensor_parallel_size` 的环境变量，即 MoE expert tensor parallel 度，用于拆分专家层参数/计算。
- **常见值**：1
- **来源环境变量**：ACTOR_ETP
- **性能影响**：文档说明：Verl best practices 建议 MoE 模型让 EP/ETP 与 TP 协同；增大 ETP 可降低单卡专家参数压力，但会引入专家层通信。
- **精度影响**：机制推断：不改变 MoE 路由或损失定义；不同并行切分可能带来轻微数值差异，资源不足导致的 batch/长度调整才会间接影响结果。
- **NPU/Ascend 证据**：未知
- **CI 看护**：部分
- **示例数**：6
- **需要子代理补证**：否
